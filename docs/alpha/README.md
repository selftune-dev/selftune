# Alpha Cohort Ops

Operator-facing assets for onboarding and running the first selftune alpha
cohort.

Use these files during the first 3-5 tester rollout:

- [alpha-invite-script.md](./alpha-invite-script.md) — concise copy for inviting
  a tester and explaining what they should expect
- [alpha-tester-checklist.md](./alpha-tester-checklist.md) — step-by-step
  checklist for installing, enrolling, verifying upload readiness, and
  confirming data flow
- [alpha-cohort-tracker.md](./alpha-cohort-tracker.md) — simple internal
  tracker for who is live, whether uploads are flowing, and what issues have
  surfaced

These documents are intentionally lightweight. The goal is not a polished
customer-success program; the goal is to remove operator ambiguity while the
cohort is still small and high-touch.
