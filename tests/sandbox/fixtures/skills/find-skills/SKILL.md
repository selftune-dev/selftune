# find-skills

Helps users discover and install agent skills from the skills.sh marketplace.

## When to Use

- User asks "how do I do X" where X might be handled by an installable skill
- User says "find a skill for...", "is there a skill that can...", "search skills"
- User wants to extend capabilities with new skills
- User mentions skills.sh or the skills marketplace

## Examples

- "Find me a skill for generating images"
- "Is there a skill that can help with web scraping?"
- "Search for a browser automation skill"
- "How do I add new skills?"
- "What skills are available for security testing?"

## Configuration

No configuration required. Works out of the box.
