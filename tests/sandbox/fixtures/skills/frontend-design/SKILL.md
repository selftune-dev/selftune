# frontend-design

Provides guidance on UI/UX design principles for building effective web interfaces.

## When to Use

- User asks about UI/UX design principles
- User needs help with layout, typography, or color choices
- User is building a web application frontend

## Examples

- "Help me design a landing page"
- "What are good color combinations for a dashboard?"
- "Review my UI layout"

## Configuration

No configuration required.
